# Prerequisites

<Callout type="default">
    We recommend that you install Java 11 or above, as anything else will not
    work.
</Callout>

You must download and install [Java](https://www.oracle.com/java/technologies/javase/jdk11-archive-downloads.html) on your computer; otherwise, any Java-based application will not work. If you don't know how to install Java, we advise you to watch this [video](https://www.youtube.com/watch?v=RBuZHg6eyIs).
